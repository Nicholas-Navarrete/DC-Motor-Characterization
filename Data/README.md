# **DATA**

This directory is where the motor characterization system dumps its data, of the sample data shown only the Arduino motor data\_20260429\_225256\_169569pts.csv is considered good. This is because the edge detection is working for the photodiode voltages. I have no idea why it is so hard to get the optical tachometer to behave...



Either way, the current appears to be noisy, and that's because it is! The current is limited to 0.4A by my power supply which is a KD3005D, its current limiting features are very noisy so the measured current is an accurate representation.
